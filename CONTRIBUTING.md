# Contributing to Agent Foundry

We welcome contributions! Here's how you can help:

## Types of Contributions

### New Team Templates
Add specialized teams for specific domains (legal, healthcare, finance, etc.)

### Enhanced Personalities
Create new agent personalities or improve existing ones

### Skills & Utilities
Add reusable skills or utility scripts

### Strategy Guides
Share decision frameworks and optimization techniques

### Bug Fixes & Improvements
Improve existing team templates, fix orchestration issues

## Development Setup

1. Clone the repository
2. Review the specs/ directory for architecture documentation
3. Use common/utilities/ for validation and testing
4. Follow existing team templates as reference

## Submission Process

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Test thoroughly
5. Submit a pull request

## Code of Conduct

Be respectful, collaborative, and constructive.
